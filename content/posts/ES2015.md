两个月前就开始准备的ES2015（原名ECMA Script 6）的技术分享，正值ES6规范正式定稿之际（完全是巧合，哈哈），在上周五的时候，给组内做了次技术分享，其中涵盖了比较能说明新特性的若干例子。

介绍了新版本规范的若干新特性：

* Arrows function
* default + reset + spread
* let + const
* Classes
* Map + Set
* Destructuring
* String
* Modules
* Iterators
* Symbols
* Enhanced Object Literals
